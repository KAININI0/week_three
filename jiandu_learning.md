# 线性回归
## 线性回归公式
模型表达式：y`=W^T * X +b
参数求解公式：W=(X^T *X)^(-1) *X^T *y
推导
1. 模型假设
线性回归假设输出与输入特征之间是线性关系：
y`=W^T * x + b
其中：
·x = (x1,x2,...,xn)^T是输入特征向量
·W = (w1,w2,...,wn)^T是权重向量
·b是偏置
·y`是预测值
2. 损失函数
采用均方误差作为损失函数，衡量预测值与真实值的差距
J(w,b)=(i/m)*sum(m,i=1)(yi-yi`)^2 =(1/m)sum(m,i=1)(yi-(w^T *xi+b))^2
其中m是样本数量，yi是第i个样本的真实值
3. 参数求解（最小二乘法）
通过梯度下降或者正规方程最小化损失函数
正规方程法：
将偏置b合并到权重向量（令w0=b，x0=1），则模型变为：y`=W^T *x
损失函数可写成矩阵形式：J（w）=（1/m）（y-Xw）^T(y-Xw)
对w求导并令导数为0：
∂J/∂W =-2/m *X^T(y-Xw)=0
解得：
w=(X^T * X)^(-1)X^T * y
